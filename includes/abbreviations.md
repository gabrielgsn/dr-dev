*[IA]: Inteligencia Artificial
*[AI]: Artificial Intelligence - Inteligencia Artificial
*[LLM]: Large Language Model - Modelo de Linguagem Grande
*[API]: Application Programming Interface - Interface de Programacao
*[MCP]: Model Context Protocol - Protocolo de Contexto de Modelo
*[IDE]: Integrated Development Environment - Ambiente de Desenvolvimento
*[CLI]: Command Line Interface - Interface de Linha de Comando
*[UI]: User Interface - Interface do Usuario
*[UX]: User Experience - Experiencia do Usuario
*[MVP]: Minimum Viable Product - Produto Minimo Viavel
*[HTML]: HyperText Markup Language
*[CSS]: Cascading Style Sheets
*[JS]: JavaScript
*[GPT]: Generative Pre-trained Transformer
*[SaaS]: Software as a Service
*[URL]: Uniform Resource Locator - Endereco web
*[deploy]: Publicar uma aplicacao para que outros possam acessar
*[prompt]: Instrucao ou comando dado a uma IA
*[token]: Unidade de texto processada pela IA (aproximadamente uma palavra)
*[open source]: Codigo aberto - software com codigo-fonte disponivel publicamente
*[no-code]: Sem codigo - plataformas que permitem criar apps sem programar
*[low-code]: Pouco codigo - plataformas que exigem programacao minima
*[frontend]: Parte visual de uma aplicacao (o que o usuario ve)
*[backend]: Parte de servidor de uma aplicacao (logica por tras)
*[framework]: Conjunto de ferramentas e padroes para construir software
*[vibe coding]: Programar usando linguagem natural com assistencia de IA
