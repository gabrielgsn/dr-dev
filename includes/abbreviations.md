*[IA]: Inteligência Artificial
*[AI]: Artificial Intelligence - Inteligência Artificial
*[LLM]: Large Language Model - Modelo de Linguagem Grande
*[API]: Application Programming Interface - Interface de Programação
*[MCP]: Model Context Protocol - Protocolo de Contexto de Modelo
*[IDE]: Integrated Development Environment - Ambiente de Desenvolvimento
*[CLI]: Command Line Interface - Interface de Linha de Comando
*[UI]: User Interface - Interface do Usuário
*[UX]: User Experience - Experiência do Usuário
*[MVP]: Minimum Viable Product - Produto Mínimo Viável
*[HTML]: HyperText Markup Language
*[CSS]: Cascading Style Sheets
*[JS]: JavaScript
*[GPT]: Generative Pre-trained Transformer
*[SaaS]: Software as a Service
*[URL]: Uniform Resource Locator - Endereço web
*[deploy]: Publicar uma aplicação para que outros possam acessar
*[prompt]: Instrução ou comando dado a uma IA
*[token]: Unidade de texto processada pela IA (aproximadamente uma palavra)
*[open source]: Código aberto - software com código-fonte disponível publicamente
*[no-code]: Sem código - plataformas que permitem criar apps sem programar
*[low-code]: Pouco código - plataformas que exigem programação mínima
*[frontend]: Parte visual de uma aplicação (o que o usuário vê)
*[backend]: Parte de servidor de uma aplicação (lógica por trás)
*[framework]: Conjunto de ferramentas e padrões para construir software
*[vibe coding]: Programar usando linguagem natural com assistência de IA
