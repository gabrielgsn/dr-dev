# Sobre o dr.dev

## O que e este site?

O **dr.dev** e uma plataforma de aprendizado para profissionais que querem
usar inteligencia artificial para criar aplicacoes -- sem precisar aprender
programacao tradicional.

O conteudo foi pensado para quem tem **pouco tempo disponivel** mas
**alta capacidade de aprendizado**. Cada modulo e direto ao ponto,
com teoria minima e muita pratica.

## Para quem e?

Este material foi criado para **profissionais experientes** que:

- Tem curiosidade sobre IA e suas aplicacoes praticas
- Querem criar ferramentas e prototipos rapidamente
- Nao tem background de programacao (mas sao tecnicos no que fazem)
- Preferem aprender fazendo do que assistindo aulas longas

!!! tip "Voce nao precisa saber programar"
    Se voce sabe escrever um e-mail claro e organizar um projeto,
    voce ja tem as habilidades fundamentais para trabalhar com IA.

## O que voce vai aprender?

Ao final da trilha, voce sera capaz de:

- **Comunicar-se com IA** de forma eficiente para obter resultados precisos
- **Escolher a ferramenta certa** para cada tipo de projeto
- **Construir aplicacoes funcionais** usando plataformas de AI coding
- **Publicar seus projetos** na internet para que outros possam usar
- **Manter-se atualizado** em um campo que evolui semanalmente

## Como usar este site

1. **Siga a ordem dos modulos** -- eles foram organizados de forma progressiva
2. **Faca os exercicios** -- a pratica e o que fixa o aprendizado
3. **Nao tenha pressa** -- cada modulo leva de 30 a 60 minutos
4. **Volte quando precisar** -- use como referencia no dia a dia

## Pre-requisitos

- Um computador com acesso a internet
- Um navegador web atualizado (Chrome, Firefox, Edge)
- Curiosidade e disposicao para experimentar
