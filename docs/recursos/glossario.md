# Glossario

Termos comuns que voce vai encontrar ao longo da trilha e no mundo de AI coding.

!!! tip "Dica"
    Ao longo do site, passe o mouse sobre termos tecnicos para ver uma explicacao rapida.

---

## A

Agente (Agent)
:   Uma IA que nao apenas conversa, mas pode executar acoes: navegar na web, editar arquivos, rodar comandos. Pense como um assistente que alem de dar conselhos, coloca a mao na massa.

API (Application Programming Interface)
:   Uma forma padronizada de dois sistemas se comunicarem. Quando o Bolt pede ao Claude para gerar codigo, ele usa uma API.

## B

Backend
:   A parte "invisivel" de uma aplicacao -- o servidor, o banco de dados, a logica de negocios. O usuario nao ve, mas e o que faz tudo funcionar.

Bolt
:   Plataforma que permite criar aplicacoes web descrevendo o que voce quer em linguagem natural. Funciona direto no navegador.

## C

Claude
:   Assistente de IA criado pela Anthropic. Conhecido por seguir instrucoes com precisao e ser transparente sobre suas limitacoes.

CLI (Command Line Interface)
:   Interface de texto onde voce digita comandos. Parece antiquado, mas e poderoso. Muitas ferramentas de desenvolvimento funcionam assim.

Cursor
:   Editor de codigo com IA integrada. Baseado no VS Code. Para quem quer mais controle sobre o codigo gerado.

## D

Deploy
:   O processo de publicar uma aplicacao para que outras pessoas possam acessar. Analogia: e a "inauguracao" do seu projeto digital.

## F

Framework
:   Conjunto de ferramentas e padroes pre-definidos para construir software. Pensando em engenharia: e como um sistema construtivo (alvenaria estrutural, steel frame) -- cada um tem suas regras e vantagens.

Frontend
:   A parte visual de uma aplicacao -- botoes, textos, menus, cores. Tudo que o usuario ve e interage.

## G

GitHub
:   Plataforma onde desenvolvedores armazenam e compartilham codigo. Tambem permite hospedar sites gratuitos (GitHub Pages).

GPT (Generative Pre-trained Transformer)
:   A arquitetura tecnica por tras de modelos como ChatGPT. Na pratica, voce nao precisa saber como funciona por dentro para usar.

## I

IDE (Integrated Development Environment)
:   Ambiente de desenvolvimento -- o "escritorio" do programador. Exemplos: VS Code, Cursor.

Iteracao
:   O processo de refinar algo em ciclos. Em AI coding: voce pede, a IA gera, voce revisa, pede ajustes, a IA refina. Cada ciclo e uma iteracao.

## L

LLM (Large Language Model)
:   O tipo de IA por tras de ferramentas como Claude e ChatGPT. Sao modelos treinados em enormes volumes de texto que aprenderam a gerar linguagem natural e codigo.

Lovable
:   Plataforma de AI coding similar ao Bolt, com foco em criar aplicacoes com design mais polido e profissional.

## M

MCP (Model Context Protocol)
:   Protocolo que permite que IAs se conectem a ferramentas externas (arquivos, bancos de dados, APIs). Pense como um "adaptador universal" para IA.

MVP (Minimum Viable Product)
:   Produto minimo viavel. A versao mais simples possivel de uma ideia que ainda funciona e entrega valor. Comece pelo MVP, depois melhore.

## N

No-code
:   Plataformas que permitem criar aplicacoes sem escrever codigo. Voce descreve o que quer, a plataforma constroi. Exemplos: Bolt, Lovable.

## O

Open source
:   Software com codigo-fonte disponivel publicamente. Qualquer pessoa pode ver, usar e contribuir.

## P

Prompt
:   A instrucao ou comando que voce da a uma IA. A qualidade do prompt determina a qualidade da resposta. E a habilidade central do AI coding.

## R

Repositorio (Repo)
:   Um "projeto" no GitHub. Contem todos os arquivos, historico de alteracoes e configuracoes de uma aplicacao.

## S

SaaS (Software as a Service)
:   Software acessado pela internet, sem precisar instalar nada. Gmail, Google Docs e Netflix sao exemplos de SaaS.

## T

Token
:   Unidade de texto processada pela IA. Aproximadamente uma palavra. Modelos de IA tem limites de tokens por conversa.

## U

UI (User Interface)
:   Interface do usuario -- todos os elementos visuais com os quais o usuario interage: botoes, formularios, menus.

URL
:   O endereco de uma pagina na internet. Exemplo: `https://gabrielgsn.github.io/dr-dev/`

## V

Vercel
:   Plataforma de hospedagem que permite publicar aplicacoes web com facilidade. Tem plano gratuito generoso.

Vibe coding
:   Programar usando linguagem natural com assistencia de IA. Voce descreve o que quer, a IA escreve o codigo. Termo popularizado em 2025.

v0
:   Ferramenta da Vercel para gerar componentes de interface. Voce descreve um elemento visual e recebe opcoes prontas.
